## рџ›  Installation

Clone the repository:

```bash
git clone https://github.com/angshuman-web/iem-poc.git
```

Navigate to the project directory:

```bash
cd iem-poc
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

Preview production build:

```bash
npm run preview
```
