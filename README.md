# 🎓 IEM Salt Lake — Website Redesign

A modern, responsive, and immersive digital experience built for the **Institute of Engineering & Management (IEM), Salt Lake Campus**.

This project reimagines the university's online presence with a premium user experience, interactive storytelling, and a student-first information architecture designed to improve discoverability, engagement, and admissions conversion.

🌐 **Live Demo:**  
https://iem-saltlake.vercel.app/

## 🛠 Installation

Clone the repository:

```bash
git clone https://github.com/angshuman-web/iem-poc.git
```

Navigate to the project directory:

```bash
cd iem-poc
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

Preview production build:

```bash
npm run preview
```
